## 八章

差分

### コマンド別

#### git diff

作業ディレクトリとインデックスの差分を表示する。

#### git diff commit

作業ディレクトリと指定したコミットの差分を表示する。

#### git diff --cached commit

インデックスにステージされた変更と指定したコミットの差分を表示する。

#### git diff commit1 commit2

指定したコミットとコミットを指定する。
